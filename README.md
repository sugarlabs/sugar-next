# Sugar Next

*A Learning Shell for Everyday Computing*

Sugar Next is a **sibling project** to the classic Sugar desktop — not a
fork and not a replacement. It lives under the
[Sugar Labs](https://www.sugarlabs.org/) umbrella and explores what a
modern, low-floor learning environment could look like using the
technologies available today (GTK4, Wayland, Python, XDG).
